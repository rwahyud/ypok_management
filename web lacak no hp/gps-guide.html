📍 Panduan GPS Real-Time

Family Location Tracker menggunakan GPS perangkat Anda untuk tracking lokasi real-time

✅ Ini BUKAN data dummy!
Aplikasi ini menggunakan HTML5 Geolocation API untuk mendapatkan koordinat GPS nyata dari perangkat Anda.

🚀 Cara Mengaktifkan GPS Real-Time

1️⃣ Izinkan Akses Lokasi Browser

Saat pertama kali membuka dashboard, browser akan menampilkan popup:

  • "Allow/Izinkan" - Klik ini untuk mengizinkan
  • "Block/Tolak" - JANGAN klik ini

🔵 Chrome/Edge

  1. Klik ikon 🔒 di address bar
  2. Pilih "Location/Lokasi"
  3. Pilih "Allow/Izinkan"
  4. Refresh halaman (F5)

🦊 Firefox

  1. Klik ikon (i) di address bar
  2. Klik "Permissions"
  3. Centang "Access Your Location"
  4. Refresh halaman (F5)

2️⃣ Aktifkan GPS di Perangkat

Windows:

  • Settings → Privacy → Location
  • Aktifkan "Location services"
  • Pastikan browser memiliki izin

Android/Mobile:

  • Settings → Location/Lokasi
  • Aktifkan "Use location"
  • Mode: High accuracy (GPS, Wi-Fi, Mobile)

3️⃣ Pastikan Sinyal GPS Baik

💡 Tips untuk GPS Akurat:
  • 📡 Gunakan di luar ruangan atau dekat jendela
  • 🏙️ Hindari area dengan bangunan tinggi yang menghalangi
  • ⏱️ Tunggu 10-30 detik untuk GPS lock pertama kali
  • 🔋 Battery saver mode bisa mengurangi akurasi GPS

🎯 Cara Kerja GPS Real-Time

📡 Teknologi yang Digunakan:

  • HTML5 Geolocation API - Browser API untuk akses GPS
  • enableHighAccuracy: true - Gunakan GPS hardware untuk akurasi tinggi
  • watchPosition() - Monitor perubahan lokasi secara continuous
  • maximumAge: 0 - Tidak gunakan cache, selalu data fresh

🔄 Update Real-Time:

  • GPS di-monitor terus-menerus saat toggle aktif
  • Lokasi dikirim ke server setiap ada perubahan
  • Peta di-update setiap 3 detik
  • Koordinat ditampilkan dengan 6 digit desimal (akurasi ±10cm)

🐛 Troubleshooting

❌ "Izin lokasi ditolak"

Solusi:

  1. Klik ikon gembok/info di address bar browser
  2. Ubah "Location" dari "Block" ke "Allow"
  3. Refresh halaman (tekan F5)
  4. Klik "Allow" pada popup yang muncul

⏱️ "GPS Timeout"

Solusi:

  • Pindah ke area dengan sinyal GPS lebih baik (luar ruangan)
  • Tunggu lebih lama (30-60 detik) untuk GPS lock
  • Restart perangkat untuk refresh GPS hardware
  • Klik tombol "🔄 Refresh Lokasi Saya"

📍 "Lokasi tidak akurat"

Penyebab & Solusi:

  • Di dalam ruangan: GPS tidak berfungsi optimal → Keluar atau dekat jendela
  • Cuaca buruk: Awan tebal bisa menghalangi → Tunggu cuaca cerah
  • Urban canyon: Gedung tinggi memantulkan sinyal → Pindah ke area terbuka
  • Battery saver: Mengurangi akurasi → Matikan sementara

🔐 Keamanan & Privacy

⚠️ Informasi Penting:
  • Lokasi Anda hanya dibagikan ke anggota keluarga dengan kode yang sama
  • Anda bisa matikan sharing kapan saja dengan toggle
  • Data lokasi tidak disimpan permanen, hanya lokasi terakhir
  • Koordinat GPS real-time dari perangkat Anda, bukan estimasi

📊 Membaca Info GPS

Info yang Ditampilkan:

  • Koordinat: Latitude, Longitude (6 desimal = akurasi ±10cm)
  • Akurasi: Radius error dalam meter (semakin kecil = lebih akurat)
  • Ketinggian: Altitude dalam meter (jika tersedia)
  • Kecepatan: Kecepatan bergerak dalam km/h (jika bergerak)
  • Timestamp: Waktu terakhir update

Tingkat Akurasi GPS:

  • ✅ < 10 meter: Sangat akurat (GPS lock baik)
  • ⚠️ 10-50 meter: Cukup akurat (GPS lock sedang)
  • ❌ > 50 meter: Kurang akurat (sinyal lemah, pindah ke luar)
