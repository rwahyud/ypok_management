## 📞 Troubleshooting

**Q: Error "Column already exists"**  
A: Normal, artinya kolom sudah ada. Skip saja atau drop query ALTER TABLE.

**Q: Statistik tidak berubah setelah edit dojo**  
A: Periksa apakah trigger berhasil dibuat. Jalankan query:
```sql
SHOW TRIGGERS WHERE `Table` = 'dojo';
```

**Q: Ingin reset/hitung ulang statistik**  
A: Jalankan query ini di phpMyAdmin:
```sql
-- Untuk semua provinsi
UPDATE provinsi p
SET 
    total_dojo = (SELECT COUNT(*) FROM dojo WHERE provinsi_id = p.id),
    total_anggota = (SELECT COALESCE(SUM(total_anggota), 0) FROM dojo WHERE provinsi_id = p.id),
    anggota_aktif = (SELECT COALESCE(SUM(anggota_aktif), 0) FROM dojo WHERE provinsi_id = p.id),
    anggota_non_aktif = (SELECT COALESCE(SUM(anggota_non_aktif), 0) FROM dojo WHERE provinsi_id = p.id);
```
